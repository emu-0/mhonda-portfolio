Light Lounge webサイト

WebサイトURL : https://mhonda-lightlounge.netlify.app

概要 overview

TECH CAMPデザイナーコースの最終課題。新規事業でコワーキングスペースを展開する株式会社LightLoungeの、集客を目的としたWebサイト制作（架空案件）。クライアントへのヒアリングからデザイン、 コーディングまでを行いました。コンセプトは「カフェ×シンプル」で、落ち着いて暖かみのある配色と、シンプルなデザインにしました。

  • 制作期間：2020/10/26〜2020/11/16（3週間）
  • 担当箇所：デザイン・コーディング
  • 使用ツール：Figma / Photoshop / HTML / CSS / jQuery
  • URL：なし

制作プロセス Process

競合分析

3C分析

クライアント、競合他社の強み、弱みを洗い出し3C分析を作成しました。これによりクライアントだけが持つ、ユーザーにとって購買要因にもなる強みは【駅から近く通いやすい、余裕のあるスペース】＋安価である、という点であることがわかりました。

ポジショニングマップ

クライアントの強みでもある要素を洗い出し、ポジショニングマップを作成しました。これによりクライアントだけが持つ、ユーザーにとって購買要因にもなる強みは【カフェのようなコワーキングスペース、低価格】である、という点であることがわかり、今回は競合に設定した2社よりも親しみやすく、落ち着きや安心感のあるデザインを狙いました。

ヒアリング内容

FVのイメージ

Webサイトの制作を行うにあたり、デザインの方向性を探るためにヒアリングを行いました。事前に要望に挙がっていたイメージの、言葉の意味するところを噛み砕き、想定される参考イメージと合わせ共有しました。

ペルソナ設定

デザインの方針を明確に、よりユーザー視点に立ったものにするため、ペルソナを設定しました。そこからペルソナのKBF(Key Buying Fuctor)＝購買決定要因を導き出し、より優先度の高いものを洗い出しました。

デザイン

全体

フォントと配色

文には、どのようなシチュエーションでも読みやすいよう、読みやすさで評価の高い「ヒラギノ角ゴシック」を使用しました。また欧文には、ペルソナの好みに合う、文字としての機能性と美しさで評価の高いOpen Sansを使用しました。配色は、緊張や不安を取り除く効果があるというオレンジ色をメインに、グレーをフォントに使用し、重厚感や落ち着きを感じられるようにしました。

工夫点

ファーストビュー

「ホッとする×シンプル」というコンセプトに沿ったデザイン。背景画像を主役にして明るく開放感のある空間を強調しました。

フロアマップ

参考画像をトレースして作成。配色で視認性UP、スタイリッシュなデザインに仕上げてサイト全体のイメージと調和させました。

アイコン

アイコンを使用することで情報を直感的に認識しやすいようにした。背景に写真を入れて強みである綺麗な空間もアピールしました。

FAQ

FAQはユーザーが知りたい答えにたどり着きやすいよう、多階層のアコーディオンにし、質問の種類ごとにカテゴリ分けをしました。
